# MuseScore
**Fast Brief:** The world's most popular free music notation software — compose, play, share, and print professional scores.

## Overview

MuseScore makes music notation accessible to everyone, from beginner students to professional composers. Its WYSIWYG interface lets you enter notes with a mouse, computer keyboard, or MIDI controller and see the engraving-quality sheet music update in real time. A built-in software synthesizer provides high-quality playback of your score with a library of orchestral, band, and electronic instrument sounds.

Beyond notation, MuseScore supports every standard score element: articulations, dynamics, lyrics, chord symbols, fret diagrams, and drum notation. The Inspector panel provides fine-grained control over every element's position, style, and playback properties. With cloud saving on MuseScore.com, you can share scores publicly, embed them on websites, and collaborate with other musicians through the online viewer and mobile apps.

## Getting Started

Download MuseScore from the official website. Launch the application and the Start Center will offer to create a new score or open a template. Choose an instrument, key signature, time signature, and tempo. Enter notes by clicking the note duration in the toolbar, then clicking the desired position on the staff. Press Space to play back your score and hear the result instantly.

## Everyday Use

1. Create a new score from File > New and select all instruments for your ensemble.
2. Press N to enter note input mode, select a duration with the number keys, and click the staff.
3. Add articulations, dynamics, and slurs from the Palettes panel on the left.
4. Use the Play Panel and Mixer to adjust instrument volumes and balance during playback.
5. Fine-tune layout with the Inspector and ensure all elements are well positioned.
6. Export as PDF, MIDI, MP3, or MusicXML via File > Export.

## Troubleshooting

1. **Playback stutters with large scores:** Increase the audio buffer size in Edit > Preferences > I/O; use the built-in Muse Sounds if installed or switch to a lighter SoundFont for smooth performance on lower-specced systems.
2. **Lyrics don't align with the correct notes:** Ensure you are in note input mode and use Space to advance to the next syllable; incorrect alignment usually results from misplacing the cursor before starting lyric entry.
3. **Score layout has too many pages with weird spacing:** Adjust the global staff scaling in Format > Page Settings and fine-tune measure widths using the stretch command or system breaks; enable Format > Style > Score > Create Multi-Measure Rests for parts.
4. **Imported MusicXML looks wrong:** Check that the MusicXML file was exported from the source application at the latest version; use Edit > Preferences > Import to toggle quantization, clef changes, and formatting import behavior.
5. **Drum notation sounds play the wrong instruments:** Select the drum staff and open Edit Drumset in the Mixer or right-click the staff; verify each notehead type maps to the correct MIDI percussion sound and adjust the mapping if necessary.
